💡 Philosophy

🎵 Vibe Coding

LLM Agent(Claude, GPT, Gemini 등)와 협업하여 코드를 작성합니다. 인간은 아이디어와 방향을 제시하고, AI가 구현을 담당합니다.

🏠 Self-Hosted First

클라우드 의존성을 최소화하고, 로컬 환경에서 모든 서비스를 운영합니다. 데이터 프라이버시와 비용 효율성을 동시에 확보합니다.

🔌 Modular & API-First

각 서비스는 독립적으로 동작하며, REST API를 통해 상호 연결됩니다. 필요한 서비스만 선택적으로 배포할 수 있습니다.

🐳 Container Native

모든 서비스는 Docker 컨테이너로 패키징되어 일관된 환경에서 실행됩니다. Docker Compose로 쉽게 배포할 수 있습니다.

📖 Documentation Driven

코드보다 문서가 먼저입니다. 명확한 스펙과 가이드를 작성한 후 구현합니다. LLM이 이해할 수 있는 컨텍스트를 항상 유지합니다.

🔄 Continuous Evolution

완벽을 추구하지 않습니다. 빠르게 배포하고, 지속적으로 개선합니다. AI와 함께 진화하는 시스템을 만듭니다.

📚 Getting Started

1

환경 준비

# Docker 및 Docker Compose 설치 확인
docker --version
docker compose version

# GPU 지원 (선택사항)
nvidia-smi
2

서비스 배포

# 원하는 서비스 클론
git clone https://github.com/vibe-homelab/vision-insight-api

# 환경 설정
cp .env.example .env

# 실행
docker compose up -d
3

API 사용

# Health Check
curl http://localhost:8000/health

# API 호출 예시
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"image_url": "..."}'
